Professional Background

History

I do not have much professional experience yet although I am currently working at the USGS office in Flagstaff Arizona. I gained much of my professional skills during group projects and my Capstone in college. For my Capstone, our team was assigned to the USGS and our client, Dr. Laszlo Kestay enjoyed the work so much that he hired us on as contractors to continue the work.

Achievements

My proudest professional moments include writing my Operating Systems program where a configuration file is fed into the program by command line and the configuration creates process control blocks for each fake batch program and then runs each program in the order that was configured. Either First Come First Serve, Shortest Job First-Nonpreemptive, Shortest Job First-Preemptive, Shortest Runtime First-Nonpreemptive, and lastly Shortest Runtime First-Preemptive. This project took many hours of grinding is C using pointers and C data structures. I'm planning on coming up with a more elgant solution soon to refresh my C skills.
I'm most proud of my current work project. The PIPS project at the USGS, which is a Server designed to bring ease to researchers' publication processes by allowing users to add icons and annotations to planetary images. This is a vital task for researchers that wish to get their work published in journals. I started out with one of my Capstone partners but I've taken the reins as the team leader, repository manager, and lead coder. The github links to both of these projects can be found in the projects tab.

Goals

Notifications can be ignored too easily, a text message tone catches someones' attention right away and that would be a far more effective means of task management.
