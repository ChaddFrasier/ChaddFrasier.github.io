My Interests

I am interested in many fields of computer science including Machine Learning, Image Processing, and Cyber-Security, as well as 3D Graphics and Game Development. I love the idea of coding on a team that creates the next big video game, and recently I have loved working with geospacial image files at the USGS. Working on the PIPS project I learned about how to manipulate images using JavaScript including converting them to Base64 strings for embedding images into SVG files. This method allows me to edit and create custom images based on metadata from the images.

Working at the USGS has sparked my interests to go back to school and dive deeper into the computer science ocean of innovation. I worked on a research project with Dr. Laszlo Kestay where we predicted the Surface Scattering variable of lunar mission data based on camera and lighting metadata. We could predict scattering values with up to 92% accuracy with relatively small training sets. The goal is to eventually be able to apply specific photometric corrections without using mathematical calculations and instead using a predictive AI model.

Recently I have been working in UI/UX design, mainly due to the fact that I am the soul programmer on the PIE project at the USGS which is the version 2 of the PIPS server. The UI design has taken me the most time but I enjoy coding the back-end much more. Since Node.js runs asynchronously, I needed to make sure that when I am processing images with GDAL the server only responds when necessary and it needs to work synchronously with the user interface. To accomplish this the server uses JavaScript Promises to guarantee synchronous execution on the server end and then I can use response design to only respond when the image processing finishes.
